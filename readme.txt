# You may wonder what is this code 
## the aim of this is to attempt to predict if a stock will go up or down in value using a Hoeffding Tree Classifier
## its very simple but a small practice to keep my brain working. it is minimal size and automatically installs required csv file. 


# How to install requirments
## Copy paste the below code to execute the application

python -m venv venv
.venv\Scripts\activate
pip install requirments.txt
cd main
python main.py
